This project showcases my ability to convert a Figma drawing into code (HTML, CSS, JS, Bootstrap, etc.).

The Stack used in this project is:
* HTML
* CSS
* SASS
* Bootstrap
* JavaScript

Project Page: https://calvinbarajas.github.io/Figma2Code/
